Conexao

nao procurou

Console serial USB: 115200 8N1, login automatico como root.

Wi-Fi do robo

nao configurado

Envia a sua rede atual para o robo (via console ja conectado, normalmente USB) e roda wpa_supplicant + udhcpc nele. Depois use o IP obtido para conectar por Wi-Fi sem cabo.

O Khepera so tem radio 2.4 GHz — confirme que essa e a banda da sua rede.

Missao

em espera

Use duracao 0 para rodar sem timeout. Para teste em bancada, comece com rodas suspensas. Iniciar roda A->B por navegacao tatil e odometria, sem mapa previo. A* usa mapa salvo apenas para comparacao.

Controle manual

teleop off

Primeiro use Enviar + compilar para instalar o modo teleop. Depois segure um botao para mover. Ao soltar, a interface envia freio; se o robo ficar 0,8 s sem comando, o modo manual para os motores sozinho.
